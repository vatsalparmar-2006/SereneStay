# 🏨 SereneStay - Hotel Management System

SereneStay is a modern Hotel Management System developed to simplify hotel operations such as room booking, customer management, reservations, billing, and administrative tasks through a user-friendly web application.

The project focuses on providing a seamless experience for both hotel administrators and customers with an efficient and scalable management solution.

---

# ✨ Features

## 👤 Guest(User) Features

* User Registration & Login
* Hotel Room Browsing
* Room Booking System
* Check Room Availability
* Booking Confirmation
* Responsive UI Design
* Contact & Inquiry System

---

## 🛠 Management Features

The admin panel includes two different roles with role-based access control:

### 👨‍💼 Manager (Admin)

Manager has full system access including:

* Manage Rooms
* Manage Customers
* Booking Management
* Reservation Tracking
* Staff Management
* Room Availability Control
* Customer Data Management
* Hotel Service Management
* Reports & Analytics
* Full Dashboard Access

### 🧑‍💻 Receptionist

Receptionist has restricted functionality including:

* Customer Check-In / Check-Out
* Booking Management
* Reservation Handling
* Customer Assistance
* Limited Dashboard Access
* Room Availability Updates

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/vatsalparmar-2006/SereneStay.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd SereneStay
```

## 3️⃣ Configure Database

Update your database connection string inside:

```bash
appsettings.json
```

Example:

```json
"ConnectionStrings": {
  "DefaultConnection": "Server=YOUR_SERVER;Database=SereneStayDB;Trusted_Connection=True;"
}
```
